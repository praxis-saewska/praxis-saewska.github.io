PWA Icon Generator for Praxis Saewska

Instructions:

  1. Click "Generate Icons" to create all required icon sizes
  2. Icons will be generated with the Praxis Saewska theme color (#00C9A7)
  3. Right-click each icon and "Save image as..." to save them
  4. Save them in the images/ directory with names: icon-72x72.png, icon-96x96.png, etc.
