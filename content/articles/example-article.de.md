## Beispielartikel

Dies ist ein **Beispiel für статью/новость**, которая будет отображаться с помощью нового шаблона `news.html`.

Sie können:

- den Titel oben ändern,
- Inhalt на любом языке написать,
- категорию (Category) поменять, например на `Blog` или `Updates`.

После этого запустите сборку сайта:

```bash
uv run pelican content -s pelicanconf.py
```

и проверьте страницу по адресу `/beispielartikel/` и список статей по ссылке «Artikel» в верхнем меню.
